Operators

Operators, as you can guess, are used to do some changes or process over variables. There are different kind of operators however, they aren’t much different than math.

Arithmetic Operators

These are simple operators that we are used to see in math. Use of plus, minus, multiplication, and division operators:

1a=3

Here we created a variable called “a” and its value is 3. You can use this value by using the variable’s name. If you want to print it, logically you should write

1a = 3+2
2b = a-1
3c = b*2
4d = c/8
5print(a,b,c,d)

You can print multiple things by using comma. There isn’t any complicated stuff with these simple operators. There are more operators in python to make things easier. Exponential (**), mod (%), floor division (//)

1a = 2**3
2b = 5%2
3c = 7//3
4print(a,b,c)

Above code means a = 2 to the power of 3 = 8 and 5 = b mod(2) which means what is the quotient when we divide 5 by 2 = 1. Floor division takes the integer part of division, 7//3 = 2.

Comparison Operators

Comparison operators can be used whether a statement is true or not. There is a great example:

1a = 5
2b = 3
3samplebool = a>b
4print(samplebool)

Above code when we write a>b, it asks “Is a greater than b?”. a is 5 and b is 3 so, answer of this question is true. Then it assigns true to samplebool variable because a>b represents True. You can try it to see the output.

There are more comparison operators as there are in math. Here is a list for comparison operators.

  • == , is equal
  • != , not(is equal) which means it will take the opposite answer of is equal. That's what ! does.
  • > , is greater
  • < , is less
  • >= , is greater or equal
  • <= , is less or equal

Assignment Operators

You have learned one of the assignment operators which is =. You could understand that it isn’t same sign with math (equal). Indeed, = operator assigns values from fright side operand to left side operand. For example, you can write a code like this if you want to increase a’s value by 1.

1a = 4
2a = a+1

Let’s analyze it. Value of right side is a+1 which equals 5. Then it assigns 5 (from right side) to a (to left side). However, there is another way to do that.

1a = 4
2a += 1

Above code does that process, takes right operand’s value and left operand’s value, does calculation with the operator + (which is left side of =) and assign the result of calculation to the left operand. Left side’s value is 4, right side’s value is 1, operator is + so result is 5, therefore a = 5.

You can do those kinds of assignments with all kind of arithmetic operators stick to equal sign. Here is a short list for assignment operators:

  • += , a+=b means a=a+b
  • -= , a-=b means a=a-b
  • *= , a*=b means a=a*b
  • /= , /= means a/=b
  • %= , %= means a=a%b
  • **= , a**=b means a=a**b
  • //= , a//=b means a=a//b

These are the fundamental operators that you should definitely be able to use. Of course, there are other operators such as bitwise operators and specific operators for python. However, they aren’t needed to be learned specifically as an operator or, they haven’t got much use.
