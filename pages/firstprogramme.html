First Program

Finally, you can write your first program! In every language first thing you learn is how to print “Hello World!” to the screen. It is not that different from writing it English. Open Python IDLE, you can find it by searching IDLE on the search bar. First window that pop up is the scripting section, write

1print("Hello World!")

Hit the enter, you wrote your first program and gave your lovely greetings to the world!

Now from file (top-left) select New File. Write your code in the new window that popped up. Then, click File>Save, name your file and save it. Now in order to execute your code click Run>Run Module or simply press F5. You saved and executed your first project. We will talk about why there are brackets after print and why we use “”. For example, we used “” because the data’s type, that we wanted to print, is a string! I know you are now a bit confused but move on to the next lesson, we will learn about variables.
